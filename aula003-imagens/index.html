Exemplo de Uso das Tags 'figure' e 'picture'

1. Exemplo de uso da tag 'figure'

A tag 'figure' é usada para agrupar uma imagem com sua legenda ou descrição.

2. Exemplo do uso da tag 'picture'

A tag 'picture' é usada para fornecer diferentes versões de uma imagem, de aordo com o tamanho da tela ou resolução.

O navegador escolhe automaticamente a versão mais adequada dad imagem.
